0 Introduction

Provides some background on the history of JavaScript and development of the standard.

1 Scope

The purpose of the document: to define the ECMAScript language.

2 Conformance

What it means for a JavaScript implementation to obey the spec. Specifically:

  • It must implement everything required by the spec.
  • It must support programs written with Unicode characters.
  • It may build on the standard library defined by the spec, either by including entirely new components, or extending existing ones.
  • It may allow syntax not defined in the spec.

3 Normative references

Standards that the spec relies on:

  • ISO/IEC 9899:1996 (C programming language)
  • ISO/IEC 10646-1:1993 (Unicode)
